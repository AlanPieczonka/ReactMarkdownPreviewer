# MarkdownPreviewer
## Made with React and Bootstrap 4
### One of the FreeCodeCamp's React Projects
### To run Previewer:
	git clone https://github.com/AlanPieczonka/ReactMarkdownPreviewer.git
	cd ReactMarkdownPreviewer
	npm install
	npm start 
### To run tests: 
	npm test test.js
